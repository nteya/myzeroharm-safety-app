# MyZeroHarm

MyZeroHarm is a mobile safety app built to improve workplace safety communication and awareness.

## About
This project was created to help workers report hazards faster, share safety information, and improve safety culture on site.

## Main Features
- Hazard reporting
- Safety alerts
- Weekly safety assessments
- AI safety guidance
- Task safety support

## Tech Stack
- React Native
- Expo
- Firebase
- Firestore

## Status
This project is still being improved and updated.
